Building now

glossmd

Active project

Word-style review comments for Markdown that lives in git — portable, append-only, no database. AI is making it easy to produce design docs and specs as Markdown, but the review experience hasn't caught up. glossmd adds select-to-comment review threads without moving the document out of git or locking comments inside a vendor's database.

The interesting bit is the protocol: comments are portable append-only data beside the document, current state is derived by replaying actions, and anchors re-locate against rendered text as the document changes.
